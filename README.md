# computer networks

## Transmission technologies

### broadcast

All machines on the network receive the same message, but only the machine with the correct address processes it.

### Multicast

A message is sent to a group of machines that have registered to receive messages for that group.

### Unicast

A message is sent to a single machine.

## Scale Classification

- PAN (Personal Area Network)
- LAN (Local Area Network)
- MAN (Metropolitan Area Network)
- WAN (Wide Area Network)
- GAN (Global Area Network)

## Network Protocols

Rules that define how machines communicate (transmit information) with each other.

## TCP/IP Model

- Layer 1 - Network Access Layer: ARP, L2TP, NDP, Ethernet.
- Layer 2 - Internet Layer: IP, ICMP, IGMP, IPsec.
- Layer 3 - Transport Layer: TCP, UDP, DCCP, ICMP, FCP.
- Layer 4 - Application Layer: HTTP, FTP, SMTP, DNS, DHCP, SSH, Telnet, SNMP, POP3, IMAP, NTP, SIP, RTP, RTCP, TLS, SSL, TLS, QUIC, BGP, NFS, LDAP, IRC, XMPP, AMQP, SMB, Zeroconf, UPnP, SLP, Bonjour, SSDP, CoAP, MQTT, STOMP

## IP Address

### IPv4

4bytes (32bits) - 2^32 = 4.294.967.296
128 - 64 - 32 - 16 - 8 - 4 - 2 - 1
0 - 255

## OSI Model (Open System Interconnection)

- Layer 1 - Physical Layer: RS-232, RJ45, V34, 100BASE-TX, SDH.
- Layer 2 - Data Link Layer: Ethernet, PPP, HDLC, ATM, IEEE 802.11, IEEE 802.15, IEEE 802.16, IEEE 802.3.
- Layer 3 - Network Layer
- Layer 4 - Transport Layer
- Layer 5 - Session Layer
- Layer 6 - Presentation Layer
- Layer 7 - Application Layer
